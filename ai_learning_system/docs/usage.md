## 启动系统

### 启动 CLI 界面

```bash
python main.py
```

启动后将看到欢迎界面：

```
╔══════════════════════════════════════════════════════════════╗
║                    🤖 AI 学习系统                            ║
║                                                              ║
║  欢迎使用 AI 学习系统！这是一个智能对话和记忆管理平台。      ║
║                                                              ║
║  输入 /help 查看可用命令                                     ║
╚══════════════════════════════════════════════════════════════╝
```

### 启动 Web API 服务

```bash
uvicorn interface.web_api:app --reload --host 0.0.0.0 --port 8000
```

参数说明：
- `--reload`: 开发模式，代码修改后自动重启
- `--host`: 绑定主机地址，`0.0.0.0` 允许外部访问
- `--port`: 服务端口，默认 8000

## 与 AI 对话

### 基本对话

直接在提示符后输入消息即可与 AI 对话：

```
🤖 AI> 你好，请介绍一下你自己

🤖 AI: 收到您的消息: 你好，请介绍一下你自己
```

### 输入历史

系统支持输入历史记录功能：
- 使用 `↑` 键浏览上一条历史输入
- 使用 `↓` 键浏览下一条历史输入
- 历史记录保存在 `~/.ai_learning_history`

### 记忆的选择性存储

系统会自动评估每条消息的重要性，只有重要性达到阈值（默认3分）的内容才会被记忆。

**重要性评分维度：**
- 内容长度（5-500字符为佳）
- 是否包含问题（疑问词、问号）
- 是否包含个人信息（第一人称）
- 是否包含数字/日期
- 情感强度

**示例：**

```
🤖 AI> 什么是机器学习？
# 重要性评分高（包含问题、专业术语）→ 会被记忆

🤖 AI> 啊啊啊
# 重要性评分低（无意义内容）→ 不会被记忆

🤖 AI> 我叫小明，今年25岁，喜欢编程
# 重要性评分高（包含个人信息、具体数据）→ 会被记忆
```

## 配置系统

### 数据库配置

数据库文件默认存储在 `storage/data/memories.db`，可以通过修改代码自定义路径：

```python
from storage.database import Database

# 自定义数据库路径
db = Database(db_path="/path/to/your/database.db")
```

### 重要性评分阈值

修改 `core/selector.py` 中的 `remember_threshold` 属性：

```python
class Selector:
    def __init__(self):
        self.remember_threshold = 3  # 修改此值调整记忆阈值
```

### 遗忘保留天数

修改 `core/memory.py` 中的 `_calculate_retention_days` 方法：

```python
def _calculate_retention_days(self, importance: int) -> int:
    if importance >= 10:
        return 3650  # 10年
    elif importance >= 7:
        return 365   # 1年
    elif importance >= 4:
        return 30    # 30天
    else:
        return 7     # 7天
```

## 常见问题

### Q: 为什么有些消息没有被记住？

A: 系统使用重要性评分机制，只有重要性达到阈值（默认3分）的内容才会被记忆。包含以下特征的内容更容易被记忆：
- 长度适中（5-500字符）
- 包含问题
- 包含个人信息
- 包含具体数据/日期
- 情感丰富

### Q: 记忆会被保存多久？

A: 根据重要性不同，保留时间也不同：
- 重要性 10：约10年（永不遗忘）
- 重要性 7-9：1年
- 重要性 4-6：30天
- 重要性 1-3：7天

长时间未被访问的记忆会被归档或遗忘。

### Q: 如何备份我的记忆数据？

A: 记忆数据存储在 SQLite 数据库文件中，直接复制以下文件即可备份：

```bash
# 备份
cp storage/data/memories.db backup/memories_backup.db

# 恢复
cp backup/memories_backup.db storage/data/memories.db
```

### Q: 系统会泄露我的隐私吗？

A: 系统内置隐私检测功能，会自动检测并加密敏感信息（如身份证号、银行卡号等）。所有数据都存储在本地，不会上传到任何服务器。

### Q: 如何清空所有记忆？

A: 删除数据库文件即可清空所有记忆：

```bash
rm storage/data/memories.db
```

### Q: CLI 界面显示乱码怎么办？

A: 确保终端支持 UTF-8 编码：

```bash
# Windows PowerShell
[Console]::OutputEncoding = [System.Text.Encoding]::UTF8

# Linux/Mac
export LANG=en_US.UTF-8
```

### Q: 如何修改 Web API 的端口？

A: 启动时指定 `--port` 参数：

```bash
uvicorn interface.web_api:app --port 8080
```

### Q: 支持哪些 Python 版本？

A: 支持 Python 3.8 及以上版本。推荐使用 Python 3.10+ 以获得最佳体验。
