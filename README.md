# ☯️ 修仙世界 - AI驱动文字修仙游戏

一个由AI驱动的沉浸式文字修仙游戏，融合传统修仙文化与现代AI技术，打造真实的修仙体验。

## ✨ 核心特性

### 🎮 真实修仙体验
- **七层境界体系**：凡人 → 练气 → 筑基 → 金丹 → 元婴 → 化神 → 渡劫 → 大乘
- **灵根系统**：天灵根、双灵根、三灵根、四灵根、五灵根，不同资质决定修炼速度
- **寿元限制**：每个境界都有寿元上限，时间紧迫，必须抓紧修炼
- **突破机制**：成功率随境界提升而降低，福缘、业力影响突破结果

### 🤖 AI驱动对话
- **智能NPC交互**：与修仙世界中的NPC进行自然对话
- **AI评判系统**：AI回复会经过天道评判，给予修为奖励
- **内容过滤**：自动过滤不符合修仙设定的AI回复

### 🌍 无限生成世界
- **动态地图生成**：探索无限生成的秘境、洞府、山脉
- **NPC独立系统**：每个NPC都有自己的目标、记忆、关系网
- **妖兽生态**：各种妖兽栖息在不同区域，等待挑战

### 🎭 丰富游戏系统
- **功法系统**：学习各种功法，提升修炼速度和战斗力
- **道具系统**：丹药、法宝、材料，支持使用AI无限生成
- **门派系统**：七玄门、黄枫谷、掩月宗等经典门派
- **事件系统**：随机遭遇各种修仙事件

## 🚀 快速开始

### 安装

```bash
# 克隆项目
git clone <repository-url>
cd ai_learning_system

# 安装依赖
pip install -r requirements.txt
```

### 运行游戏

```bash
# 启动游戏
python main.py
```

## 🏗️ 技术架构

### 核心设计
- **AI学习系统**：选择性记忆、智能遗忘、价值判断
- **NPC独立系统**：每个NPC有自己的目标、需求、记忆、关系
- **无限生成系统**：基于规则和AI的混合生成方案

### 数据流
```
用户输入 → CLI → 游戏系统 → AI引擎 → 内容过滤 → 输出显示
                ↓
            数据库持久化
```

## 🎯 游戏特色

### 真实的修仙体验
- 资质决定修炼速度，天灵根万中无一
- 突破有失败风险，可能受伤甚至死亡
- 寿元有限，必须在寿元耗尽前突破
- 死亡后可转世重生，继承部分修为

### 生动的修仙世界
- 每个地点都有独特的NPC生态
- NPC会自主活动、社交、修炼
- 随机事件让修仙之路充满变数
- 探索未知区域发现奇遇

### 智能AI交互
- 与AI道友论道，获得修为奖励
- AI会根据修仙设定进行内容过滤
- 对话内容会被记忆，影响后续交互

## 🛠️ 技术栈

- **Python 3.8+**: 核心编程语言
- **SQLite**: 本地数据存储
- **llama-cpp-python**: 本地LLM推理（可选）
- **FastAPI**: Web API框架（可选）

## 📝 配置说明

### 模型配置
将GGUF格式的模型文件放入 `models/` 目录，游戏会自动加载。

### 游戏配置
修改 `config/game_config.py` 可调整游戏数值：
- 修炼速度
- 突破成功率
- 寿元上限
- 事件触发概率
