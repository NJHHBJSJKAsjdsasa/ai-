# 扩展游戏内容指南

本指南详细介绍如何在修仙世界游戏中添加新地点、新功法、新道具和新NPC类型。

## 1. 添加新地点

### 1.1 配置位置
在 [game_config.py](file:///workspace/ai_learning_system/config/game_config.py) 的 `GAME_CONFIG["world"]["locations"]` 列表中添加新的地点配置。

### 1.2 配置格式

```python
{
    "name": "地点名称",
    "description": "地点描述",
    "realm_required": 境界要求（0-9）,
    "danger_level": "危险等级（安全/普通/危险/绝境）",
    "location_type": "地点类型（区域/城镇/门派/野外/秘境）",
    "parent_location": "父地点名称（如果有）"
}
```

### 1.3 示例：添加新地点

```python
# 在 GAME_CONFIG["world"]["locations"] 列表中添加
{
    "name": "青丘山",
    "description": "青丘山，传说中狐仙居住之地，山清水秀，灵气充沛。",
    "realm_required": 2,
    "danger_level": "普通",
    "location_type": "野外",
    "parent_location": "镜州"
}
```

## 2. 添加新功法

### 2.1 配置位置
在 [techniques.py](file:///workspace/ai_learning_system/config/techniques.py) 的 `TECHNIQUES_DB` 字典中添加新的功法配置。

### 2.2 配置格式

```python
"功法名称": Technique(
    name="功法名称",
    description="功法描述",
    technique_type=TechniqueType.类型（CULTIVATION/COMBAT/MOVEMENT/SECRET/FORBIDDEN）,
    element=ElementType.属性（NONE/METAL/WOOD/WATER/FIRE/EARTH/WIND/THUNDER/ICE/DEMON/IMMORTAL）,
    realm_required=所需境界等级,
    effects=["效果1", "效果2"],
    learning_difficulty=学习难度（1-100）,
    cultivation_speed_bonus=修炼速度加成,
    combat_power_bonus=战斗力加成,
    special_abilities=["特殊能力1", "特殊能力2"],
    origin="功法来源",
    is_combat_skill=是否为战斗技能,
    mana_cost=法力消耗,
    damage_type=DamageType.伤害类型（PHYSICAL/MAGIC/TRUE）
)
```

### 2.3 示例：添加新功法

```python
"狐仙决": Technique(
    name="狐仙决",
    description="青丘山狐仙传承功法，修炼后可化形变化，魅惑敌人。",
    technique_type=TechniqueType.CULTIVATION,
    element=ElementType.WOOD,
    realm_required=2,  # 筑基期
    effects=["化形变化", "魅惑敌人", "速度提升"],
    learning_difficulty=65,
    cultivation_speed_bonus=0.15,
    combat_power_bonus=0.25,
    special_abilities=["狐火", "魅惑术", "化形"],
    origin="青丘山狐仙传承",
    is_combat_skill=True,
    mana_cost=15,
    damage_type=DamageType.MAGIC
)
```

## 3. 添加新道具

### 3.1 配置位置
在 [items.py](file:///workspace/ai_learning_system/config/items.py) 的 `ITEMS_DB` 字典中添加新的道具配置。

### 3.2 配置格式

```python
"道具名称": Item(
    name="道具名称",
    description="道具描述",
    item_type=ItemType.类型（TREASURE/PILL/MATERIAL/SPIRIT_STONE/BOOK/CONSUMABLE）,
    rarity=ItemRarity.稀有度（COMMON/UNCOMMON/RARE/EPIC/LEGENDARY/MYTHIC）,
    effects=["效果1", "效果2"],
    value=价值（灵石）,
    stackable=是否可堆叠,
    max_stack=最大堆叠数量,
    usable=是否可使用,
    level_required=所需等级,
    origin="道具来源"
)
```

### 3.3 示例：添加新道具

```python
"狐仙内丹": Item(
    name="狐仙内丹",
    description="青丘山狐仙修炼而成的内丹，蕴含强大的木属性灵气。服用后可大幅提升修为，并有几率获得魅惑能力。",
    item_type=ItemType.MATERIAL,
    rarity=ItemRarity.EPIC,
    effects=["大幅提升修为", "获得魅惑能力", "木属性亲和"],
    value=80000,
    stackable=True,
    max_stack=5,
    usable=True,
    level_required=3,
    origin="青丘山狐仙掉落"
)
```

## 4. 添加新NPC类型

### 4.1 配置位置
NPC类型主要通过 [models.py](file:///workspace/ai_learning_system/game/npc/models.py) 中的 `NPCData` 数据类来定义属性，然后在生成NPC时设置不同的属性值来创建不同类型的NPC。

### 4.2 自定义NPC属性
可以在 `NPCData` 类中添加新的属性来支持新的NPC类型：

```python
@dataclass
class NPCData:
    # 现有属性...
    
    # 新添加的属性
    race: str = "人类"  # 种族
    special_trait: str = ""  # 特殊特质
    unique_ability: str = ""  # 独特能力
```

### 4.3 示例：生成新类型NPC

在生成NPC时，可以通过设置不同的属性值来创建新类型的NPC：

```python
# 生成狐仙NPC
fox_fairy_npc = NPCData(
    id="npc_fox_fairy_1",
    name="白灵",
    dao_name="青丘仙子",
    age=500,
    lifespan=1000,
    realm_level=3,  # 金丹期
    sect="青丘山",
    personality="温柔善良",
    occupation="狐仙",
    location="青丘山",
    sect_type="中立",
    sect_specialty="魅惑术",
    morality=50,  # 善良
    race="狐族",
    special_trait="化形能力",
    unique_ability="魅惑术"
)
```

## 5. 测试新内容

添加新内容后，可以通过以下步骤测试：

1. 重启游戏
2. 使用 `/前往 新地点名称` 命令测试新地点
3. 使用 `/学习 新功法名称` 命令测试新功法
4. 使用 `/背包` 命令查看新道具（需要先获得）
5. 使用 `/npcs` 命令查看新类型NPC（需要在对应地点生成）

## 6. 最佳实践

- **地点**：确保新地点的境界要求合理，与现有地点形成等级梯度
- **功法**：平衡功法的效果和学习难度，避免过于强大
- **道具**：根据稀有度设置合理的价值和效果
- **NPC**：为不同类型的NPC设置独特的个性和能力，增加游戏多样性
